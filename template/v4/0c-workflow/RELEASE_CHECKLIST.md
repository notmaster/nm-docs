# Release Checklist

本文档用于首版上线或重要版本发布前检查。0 到 1 开发阶段不强制远端 CI/CD，但正式上线前必须完成本清单。

## 基础状态

- [ ] `main` 是当前稳定发布分支。
- [ ] `dev` 已包含本次准备发布的全部内容。
- [ ] `0b-goals/ROADMAP.md` 中没有未完成的阻断阶段（`blocked` 或半途的 `in_progress`）。
- [ ] ROADMAP 待人工验收清单已逐项处理，或管理员明确接受遗留项。
- [ ] `0a-docs/DECISIONS.md` 已记录影响发布的重要决策。

## 本地验证

- [ ] `./0d-scripts/check-workflow.sh` 通过。
- [ ] `./0d-scripts/verify.sh` 通过。
- [ ] 各阶段 `Verify:` 命令在发布分支上仍然通过。
- [ ] 项目类型对应的附加测试通过。
- [ ] 失败或跳过的检查已记录原因。

## 产品验收

- [ ] Spec 的首版范围（非目标之外的内容）已完成。
- [ ] Spec 的最终验收标准逐项通过。
- [ ] 关键页面完成桌面端和移动端人工验收。
- [ ] 文案、图片、链接、表单和空状态已检查。

## 安全审查

- [ ] 已使用 `0a-docs/0c-prompts/security-review.md` 完成至少一轮安全审查。
- [ ] 上线阻断级安全问题已修复。
- [ ] 密钥、Webhook secret、管理 token 没有进入仓库或前端包。
- [ ] 管理后台和敏感接口有服务端鉴权。
- [ ] 生产环境关闭调试开关、详细错误输出和测试账号。

## 部署配置

- [ ] 目标平台、项目名、环境和分支已确认。
- [ ] 环境变量已按生产环境配置。
- [ ] 域名、DNS、HTTPS 和重定向规则已确认。
- [ ] 数据库、对象存储、队列、邮件或第三方服务权限已确认。
- [ ] 免费额度、计费风险和平台限制已确认。

## 上线后 Smoke Test

- [ ] 首页或入口页可访问。
- [ ] 核心用户路径可完成。
- [ ] 管理后台可登录并完成关键操作。
- [ ] 表单、API、上传、邮件或其他外部服务按需验证。
- [ ] 日志和错误监控没有出现阻断级错误。

## 回滚方案

- [ ] 明确上一稳定版本或可回滚分支。
- [ ] 明确回滚命令或平台操作路径。
- [ ] 数据迁移有回滚或备份方案。
- [ ] 管理员知道回滚负责人和触发条件。
